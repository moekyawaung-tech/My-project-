# My Wiki Project

This is a simple wiki-style static website built with HTML, CSS, and JavaScript.

## Features
- Home page
- Multiple wiki pages
- Search box
- Responsive layout
- GitHub Pages ready

## Run Locally
Just open `index.html` in your browser.

## Deploy to GitHub Pages
1. Create a new GitHub repository
2. Upload all files
3. Go to **Settings > Pages**
4. Select branch: `main` and folder: `/root`
5. Save

Your site will be live on:
`https://moekyawaung-tech.github.io/My-project/`
